\section{符号表}\label{sec:math-tables}

有几个注意事项：
\begin{enumerate}
  \item \textcolor{blue}{蓝色}的命令依赖 \pkg{amsmath} 宏包（非 \pkg{amssymb} 宏包）；
  \item 带有角标\lsym 的符号命令依赖 \pkg{latexsym} 宏包。
\end{enumerate}
